# tocaTintas

`tocaTintas` é um projeto para macOS que permite reproduzir e transmitir áudio via AirPlay. Ele inclui um visualizador de histograma em tempo real e suporte a diversas bibliotecas de áudio.

## Requisitos
- macOS 12.4 ou superior
- Xcode com CocoaPods instalado

## Como compilar
1. Instale as dependências com `pod install`.
2. Abra `tocaTintas.xcworkspace` no Xcode.
3. Compile e execute o projeto.
4. O projeto inclui um plugin Quick Look para visualizar capas de arquivos FLAC, embutido automaticamente.
